# vue-trivia

View live: https://tho-daskalakis.github.io/vue-trivia/

This trivia project utilizes the Open Trivia Database to fetch questions and
provide the user feedback about their answers, as well as a score at the end of
each game. Visit [Open Trivia DB](https://opentdb.com/) for more info on the
API.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
